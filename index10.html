Teaching Style

Project Based

Across all months

Focused on real-world problem solving and guided project building.

Module 1

Advanced AI Concepts

Week 1: Revision of AI and ML basics plus introduction to advanced concepts.
Week 2: Supervised and unsupervised learning in simple language.
Week 3: Bias in AI with real-life examples.
Week 4: Accuracy concept and evaluation basics.

Learning outcomes

  • Explain advanced AI terms in simple form.
  • Differentiate supervised and unsupervised learning.
  • Recognize bias and accuracy as model quality ideas.
  • Discuss why evaluation matters in AI systems.
Module 2

Python for AI (Basic)

Week 5: Python revision and lists.
Week 6: Functions.
Week 7: Working with simple datasets.
Week 8: Mini practice in data handling.

Learning outcomes

  • Use lists and functions in basic Python tasks.
  • Work with small datasets in a structured way.
  • Apply coding for AI-related classroom activities.
  • Build confidence in text-based programming.
Module 3

AI Project Lifecycle

Week 9: Problem identification.
Week 10: Data collection and preparation.
Week 11: Model idea and how AI learns.
Week 12: Result and improvement.

Learning outcomes

  • Follow the AI project cycle from idea to improvement.
  • Frame a real-world problem clearly.
  • Understand data as the base for model learning.
  • Think in structured steps for project design.
Module 4

Tools & Applications

Week 13: Chatbot building basics.
Week 14: Image AI tools and demos.
Week 15: Voice AI tools.
Week 16: Basic automation and simple workflows.

Learning outcomes

  • Use practical AI tools with purpose.
  • Understand how chat, image, and voice tools work.
  • Create simple automation workflows.
  • Choose a tool based on the task.
Module 5

AI + Society

Week 17: AI in jobs and future careers.
Week 18: Ethics in AI.
Week 19: Copyright and data ownership.
Week 20: Responsible AI usage.

Learning outcomes

  • Understand the wider social impact of AI.
  • Recognize ethical issues, copyright, and ownership concerns.
  • Use AI responsibly in school and daily life.
  • Discuss how AI affects careers and future learning.
Module 6

Capstone Project Start

Week 21: Project selection.
Week 22: Planning with problem and solution.
Week 23: Data collection or tool selection.
Week 24: Initial development.

Learning outcomes

  • Choose an appropriate project idea.
  • Plan the project in a simple structured way.
  • Gather data or select the right tool.
  • Start building the solution with guidance.
Module 7

Capstone Project Completion

Week 25: Development.
Week 26: Testing and improvements.
Week 27: Finalization.
Week 28: Presentation, evaluation, and certificate.

Learning outcomes

  • Complete an AI-based capstone project.
  • Test and refine the project through feedback.
  • Present the work clearly and confidently.
  • Reflect on learning and future improvement.
Capstone Ideas

Sample project ideas

Smart attendance system

Track attendance using a simple AI-based concept or workflow.

Crop disease detection

Create a basic demo or concept for identifying crop issues.

AI study assistant

Design a tool that helps with study support or revision.

Traffic or pollution solution

Build a problem-solving concept for civic or environmental needs.

Overall Outcomes

Student learning outcomes

Advanced AI understanding

Understand ML types, bias, accuracy, and model performance basics.

Programming skills

Use Python lists, functions, and simple datasets in AI tasks.

Project development

Follow the AI project lifecycle from problem to result and improvement.

Practical AI skills

Build chatbots, use image and voice tools, and create automation.

Real-world awareness

Learn AI impact, ethics, copyright, and responsible usage.

Career readiness

Gain foundational exposure, build a portfolio project, and present confidently.
